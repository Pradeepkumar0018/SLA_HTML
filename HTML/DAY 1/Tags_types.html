Closing tag : A tag that has both an opening part and a closing part. It is used to contain text or other content inside it. Example: A paragraph tag contains text between opening and closing tags. Self-closing tag: A tag that closes by itself in a single line. It does not need a separate closing tag because it usually does not store content inside it. Example: A line break tag is used to move text to the next line. Empty tag: A tag that has no content inside it. It performs a specific task directly. Example: An image tag displays an image without containing text inside it.
