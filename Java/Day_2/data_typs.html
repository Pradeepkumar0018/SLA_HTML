Data Types in JavaScript

What is Data Type?

A Data Type defines the type of value that a variable can store. JavaScript uses different data types to store text, numbers, true/false values, and more.

Why Data Types are Important?

Data types help JavaScript understand what kind of value is stored inside a variable. They improve code readability, reduce errors, and allow developers to perform appropriate operations on data.

  • Improves code quality
  • Prevents unexpected errors
  • Makes debugging easier
  • Helps in data validation
  • Enhances program performance

Primitive Data Types

Primitive Data Types are basic data types in JavaScript. They store a single value and are immutable.

Primitive Data Types List

  • String
  • Number
  • Boolean
  • Undefined
  • Null
  • Symbol
  • BigInt

1. String

A String is used to store text or characters. Strings are written inside single quotes (' ') or double quotes (" ").

Example:

let studentName = "Naveen";
console.log(typeof studentName, studentName);

// Output:
// string Naveen

2. Number

A Number is used to store numeric values such as integers and decimal numbers.

Example:

let age = 25;
console.log(typeof age, age);

// Output:
// number 25

3. Boolean

A Boolean stores only two values: true or false.

Example:

let isStudent = true;
console.log(typeof isStudent, isStudent);

// Output:
// boolean true

4. Undefined

A variable that is declared but not assigned a value has the Undefined data type.

Example:

let city;
console.log(typeof city, city);

// Output:
// undefined undefined

5. Null

Null represents an intentional absence of value.

Example:

let data = null;
console.log(typeof data, data);

// Output:
// object null

6. Symbol

A Symbol is used to create unique identifiers. Every Symbol value is unique.

Example:

let id = Symbol("unique");
console.log(typeof id, id);

// Output:
// symbol Symbol(unique)

7. BigInt

BigInt is used to store very large integers that are bigger than the Number data type can safely handle.

Example:

let population = 12345678901234567890n;
console.log(typeof population, population);

// Output:
// bigint 12345678901234567890n

Non-Primitive Data Types

Non-Primitive Data Types can store multiple values and complex data. These are mutable and are stored by reference.

  • Object
  • Array
  • Function

Example:

let student = {
    name: "Naveen",
    age: 20
};

let colors = ["Red", "Blue", "Green"];

function greet() {
    console.log("Welcome");
}

Conclusion

JavaScript provides different data types to store various kinds of information. Understanding data types is one of the most important fundamentals of JavaScript programming and helps developers write efficient and error-free applications.
