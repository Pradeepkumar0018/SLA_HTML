Javascript questions

1. What is a variable??

Ans: Varibles are containers used to store data or information in a program Example: let name = "Bala"; here name is the variable

2. What is a Declaration??

Ans: Declaration means creating a variable. Example: let age; here, the variable age is declared but no value is assigned.

3. What is Initialization??

Ans: Initialization means assigning a value to the variable Example: let name = "Naveen"; here name is the variable and the Valuejohn is assigned or stored in it

4. Three Types of Keywords Used for Variables

1. Var
Older way of creating variables.

var city = "Chennai";
2. Let
Modern way of creating variables.

let age = 22;
3. Const
Used for fixed values.

const country = "India";

5. What is the Compiler in JavaScript and its Name?

A compiler is a program that converts JavaScript code into machine code so the computer can understand and execute it.

JavaScript Engine Names:

  • V8 Engine → Google Chrome
  • SpiderMonkey → FireFox
  • JavaScriptCore → Safari

6. What are Printing Statements?

Printing statements are used to display output. Examples: console.log("Hello World");   alert("Hello");   document.write("Hello");
