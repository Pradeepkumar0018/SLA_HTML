1. Introduction to CSS

CSS stands for Cascading Style Sheets. It is used to style HTML documents and control colors, layouts, spacing, fonts, and animations.

CSS separates the presentation layer from the content layer, making websites easier to maintain and update.

Why CSS is Important?

  • Improves website appearance
  • Makes pages responsive
  • Provides consistency
  • Reduces code repetition
  • Enhances user experience

2. History of CSS

CSS was introduced in 1996 by the World Wide Web Consortium (W3C).

Before CSS, all styling was written directly inside HTML. This made websites difficult to manage.

Version Year
CSS1 1996
CSS2 1998
CSS3 1999+

3. Types of CSS

Inline CSS

Inline CSS is applied directly within an HTML element.

Internal CSS

Internal CSS is written inside the style tag.

External CSS

External CSS is stored in a separate file.

4. CSS Syntax

CSS follows a simple syntax consisting of selectors and declarations. 

selector {
    property: value;
}

Example: 

h1 {
    color: blue;
}

5. CSS Selectors

Element Selector

Selects all elements of a given type.

Class Selector

Selects elements with a specific class.

ID Selector

Selects a unique element using an ID.

Universal Selector

Selects all elements on a page.

Selector Example
Element p
Class .box
ID #header
Universal *
