CSS Makes Webpages Attractive

CSS stands for Cascading Style Sheets. It is used to design and style HTML webpages.

CSS helps to change colors, fonts, spacing, and layouts of webpages.

Types of CSS

Internal, External, and Inline CSS

External CSS is written in a separate file and linked with HTML.

Internal CSS uses the style tag inside the head section of HTML.
